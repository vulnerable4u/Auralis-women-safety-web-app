System Overview

The Women Safety Assistant is a real-time safety monitoring system that uses advanced AI and computer vision to detect threats and provide immediate assistance.

System Workflow

1. User Registration

User registers with 4-10 emergency contacts. System stores user data securely.

→

2. Camera Activation

Dual camera system activates (front + back). Real-time video feed starts.

→

3. Threat Monitoring

System continuously monitors:

  • Speech Analysis (MFCC): Emotion detection using trained ML model
  • Motion Detection: Movement pattern analysis
  • Fusion Model: Combines speech + motion scores

→

4. Master Model Analysis

Master model analyzes fusion data:

  • Combined threat score calculation
  • Threat level classification (SAFE/LOW/MEDIUM/HIGH/CRITICAL)
  • Safe place recommendation based on threat level

→

5. Safe Place Analysis

System analyzes safe places within 1.5km:

  • Person presence analysis
  • Public accommodation priority
  • Police stations and hospitals prioritized
  • Distance and accessibility scoring

→

6. AI Assistant Response

AI chatbot:

  • Monitors threat levels continuously
  • Auto-alerts emergency contacts (if enabled)
  • Suggests 2 nearest safe places
  • Provides real-time safety recommendations

→

7. Emergency Response

When threat detected:

  • Push notifications to all emergency contacts
  • Location sharing with contacts
  • Nearest safe places with directions
  • SOS activation available
